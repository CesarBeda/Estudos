O Objeto Date

O Objeto date permite-nos exibir datas e horas em nossas páginas. Ele contém uma grande variedade de métodos, por meio dos quais podemos exibir a data atual na nossa página, fazer cálculos envolvendo horas e dias transcorridos, entre outras coisas. O primeiro procedimento antes de acessar os métodos desse objeto é criá-lo, ou seja, criar uma instância do objeto.

var Hoje = new Date();

Criamos uma variável chamada "Hoje" e atribuimos a ela o objeto date.

Dia do mês

<Script Language="JavaScript">
var Hoje = new Date();
var Dia = Hoje.getDate();
window.alert( "Hoje é dia : " + Dia );
</Script>

Mês do ano

<Script Language="JavaScript">
var Hoje = new Date();
var Mes = Hoje.getMonth()+1;
window.alert( "Estamos no mês : " + Mes );
</Script>

Repare que foi adicionado o valor (1) no método getMonth(), pois este método retorna valores de 0 a 11 para representar os meses de Janeiro a Dezembro, respectivamente. Podemos criar uma array com os nomes dos meses e trabalhar com os índices relacionados com o getMonth().

<Script Language="JavaScript">
var Hoje = new Date();
var Mes = Hoje.getMonth();
var Nome_mes = new Array ("Janeiro","Fevereiro","Março","Abril","Maio","Junho","Julho","Agosto","Setembro","Outubro","Novembro","Dezembro");
window.alert( "Estamos no mês : " + Nome_mes[Mes] );
</Script>

Retornando o ano

<Script Language="JavaScript">
var Hoje = new Date();
var Ano = Hoje.getFullYear();
window.alert( "Estamos no ano : " + Ano );
</Script>

Dia da Semana

<Script Language="JavaScript">
var Hoje = new Date();
var Dia_Semana = Hoje.getDay();
window.alert( "Dia da Semana : " + Dia_Semana );
</Script>

Horas, minutos e segundos

<Script Language="JavaScript">
var Hoje = new Date();
var Hora = Hoje.getHours();
var Minuto = Hoje.getMinutes();
var Segundo = Hoje.getSeconds();
var H = Hora + ":" + Minuto + ":" + Segundo;
window.alert( "São exatamente " + H + " horas." );
</Script>
