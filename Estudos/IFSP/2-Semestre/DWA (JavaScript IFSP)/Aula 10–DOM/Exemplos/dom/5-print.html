Método print

Impressão da página

O método print pertence ao objeto window, mas foi inserido neste capítulo porque estamos estudando o objeto document e a impressão da página tem tudo a ver com este objeto. O seguinte script cria um botão para imprimir a página.

<form> <input type="button" value="Imprimir" onclick="imprimir()"> </form>

<Script language="JavaScript">
function imprimir()
{
window.print();
}
</Script>
