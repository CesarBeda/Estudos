A Função setInterval

Esta função é utilizada para definir intervalos de tempo na execução de funções. A diferença entre a setInterval e a setTimeout é clara: setTimeout processa a função uma vez, após o atraso de tempo, já setInterval, irá processar a função, várias vezes, separadas por um determinado tempo. Este tempo é definido em milésimos de segundos, ou seja, se desejarmos um atraso intervalo de 2 segundos entre a execução da função, deveremos declarar o tempo como sendo 2000 (milisegundos). Veja o exemplo prático.


Veja o código fonte

<Script Language="JavaScript">

var intervalo = setInterval('mensagem()',5000);

function mensagem() {
window.alert( "Já se passaram 5 segundos .." ); }

</Script>;

Você pode interromper a função setInterval a qualquer tempo, fazendo uso do método clearInterval() que acessa a função por meio de uma ID e cancela automaticamente a execução.

<input type="button" value="Cancela" onClick="intervalo=window.clearInterval(intervalo)">
