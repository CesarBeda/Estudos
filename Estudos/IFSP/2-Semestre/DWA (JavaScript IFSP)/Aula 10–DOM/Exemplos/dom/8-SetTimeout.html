A Função setTimeout

setTimeout é um método do objeto window que permite disparar uma ação após um determinado tempo. Por exemplo, podemos chamar uma função qualquer após um limite de tempo determinado. window.alert( "Expressão", Tempo ); Expressão é a função que será processada e Tempo é o tempo de atraso para que a função seja executada. Este tempo é definido em milésimos de segundos, ou seja, se desejarmos um atraso de 2 segundos, deveremos declarar o tempo como sendo 2000 (milisegundos). Veja um exemplo prático, clicando no botão e aguardando 3 segundos....

<Script Language="JavaScript">
function mensagem() {
window.alert( "Já se passaram 3 segundos .." ); }
</Script>

Para testar esta função, faça a sua chamada no corpo da página, por exemplo com o seguinte código

<input type="button" value="Teste" onClick="setTimeout('mensagem()',3000)">
